Sobre o Video-Saver

Video-Saver é um projeto pessoal para organizar vídeos e conteúdos de aprendizado. Permite criar grupos, salvar links, adicionar notas e marcar vídeos assistidos. Projeto open-source. Código disponível no repositório público.

Funcionalidades chave:

  • Adicionar e editar vídeos por URL.
  • Categorizar por grupos e tags.
  • Marcar vídeos como assistidos.
  • Pesquisa por título e descrição.
  • API REST simples para integração.
